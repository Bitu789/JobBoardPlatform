# Job Board Platform

## Description

A full-stack Job Board Platform developed using React, Django REST Framework, and PostgreSQL.

## Technologies Used

* React.js
* Django REST Framework
* PostgreSQL
* JWT Authentication
* Axios
* Vite

## Features

* User Registration
* User Login
* View Available Jobs
* Apply for Jobs
* Dashboard
* Django Admin Panel
* REST APIs

## Project Structure

```
JobBoardPlatform/
│
├── backend/
├── frontend/
└── README.md
```

## How to Run

### Backend

```
cd backend
venv\Scripts\activate
python manage.py runserver
```

### Frontend

```
cd frontend
npm install
npm run dev
```
